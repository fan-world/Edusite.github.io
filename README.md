## 📝 Про проєкт
**Edusite** — це сучасний та адаптивний веб-сайт для освітньої платформи. Проєкт створений для демонстрації навичок верстки та представлення курсів для онлайн-навчання.

## ✨ Основні розділи
- **Головний екран (Hero section):** Привітання та заклик до дії.
- **Про нас:** Короткий опис переваг платформи (Community, Expert Teachers тощо).
- **Каталог курсів:** Перелік доступних навчальних програм із картками.
- **Чому ми:** Блок із ключовими особливостями сервісу.
- **Контакти та Футер:** Швидкий доступ до навігації та соціальних мереж.

## 🛠 Технології
У проєкті використані стандартні інструменти веб-розробки:
- **HTML5** — структура контенту.
- **CSS3** — стилізація та дизайн.
- **Адаптивність** — сайт коректно відображається на різних пристроях.
